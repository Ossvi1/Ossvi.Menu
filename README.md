## 🚀 ¿Cómo ejecutar?
1. Clona el repositorio
2. Instala las dependencias con:
   ```bash
   pip install -r requirements.txt
   ```
3. Ejecuta la app:
   ```bash
   python menusflask/main.py
   ```
